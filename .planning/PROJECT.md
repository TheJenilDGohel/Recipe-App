# Project: Context-Aware Recipe Discovery App

## Core Value
A production-grade Flutter application that provides an intelligent, offline-capable recipe discovery experience by leveraging time and location data to suggest relevant meals, ensuring users always have cooking inspiration regardless of connectivity.

## Core Constraints
- Framework: Flutter
- API: TheMealDB
- Architecture: Offline-first with robust state management (Bloc/Riverpod/MobX)
- Delivery: Automated CI/CD with APK releases via GitHub Actions
- Performance: Perceived performance via Shimmers and smooth transitions
