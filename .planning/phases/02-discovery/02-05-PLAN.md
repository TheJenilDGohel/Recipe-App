    - Use a Timer to delay the API call.
    - Cancel the Timer if the provider is disposed (using ref.onDispose).
    - Ensure ref.keepAlive() (or ref.cacheFor logic) is only called AFTER a successful fetch, or handle it such that stale providers for partial queries are properly cleaned up.
