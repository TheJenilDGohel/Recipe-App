# Phase 6: UI Refinement & Reference Alignment - Context

## Goal
Overhaul the visual identity of the app to match the modern, minimalist aesthetic provided in the user reference.

## Scope
- **Theme Overhaul**: Update `ThemeData` for sophisticated typography and color harmony.
- **HomeScreen Refinement**: Modernize the Search bar and Contextual Carousel.
- **Card Redesign**: Update `RecipeCard` and `MiniRecipeCard` for high-fidelity aesthetics.
- **Detail Screen Polish**: Enhance the `RecipeDetailScreen` layout for better readability and visual impact.

## Success Criteria
1. Visual parity with the reference image's minimalist and premium "feel".
2. Consistent application of new spacing and border-radius rules (likely 16-24dp).
3. Improved optical alignment across all features.

## Technical Tasks
- Implement a custom `TextTheme`.
- Refine `BoxShadow` and `BorderRadius` constants.
- Update `RecipeCard` stack layout for better text legibility on images.
